# ChatBot Assistente Jurídico — Resumo de Responsabilidades

Esta API fornece funcionalidades centrais para um assistente jurídico conversacional. Suas responsabilidades principais são:

- Receber e processar mensagens de chat entre usuários e o assistente.
- Gerenciar sessões/conversas, mantendo contexto entre mensagens.
- Integrar com um modelo de linguagem (LLM) para gerar respostas e rascunhos de documentos jurídicos.
- Realizar buscas semânticas/por similaridade em base de conhecimento ou documentos indexados.
- Autenticar requisições (API key / token) e aplicar controles básicos de uso e segurança.
